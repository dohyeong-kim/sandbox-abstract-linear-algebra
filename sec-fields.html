    Section 5.1 체

    선형대수학에서 벡터 공간을 정의할 때, 우리는 스칼라라고
    스칼라:scalar
    불리는 원소들을 사용하여 벡터의 길이를 늘이거나 줄입니다. 앞선 강의에서 스칼라의 집합으로 실수 집합 \(\mathbb{R}\)이나 복소수 집합 \(\mathbb{C}\)를 주로 언급했지만, 수학적으로 “스칼라”의 역할을 수행하기 위해서는 덧셈, 뺄셈, 곱셈, 그리고 0이 아닌 수로 나누는 나눗셈이 자유롭게 가능해야 합니다. 이러한 네 가지 사칙연산이 완벽하게 정의되고 우리가 익숙한 대수적 법칙들을 만족하는 수학적 구조를 추상화하여 체라고 부를 것입니다.
    체:field
    선형대수학의 정리 대부분은 스칼라가 어떤 체에서 왔는지에 상관없이 보편적으로 성립하며, 일부 정리는 그 성립가능성이나 참거짓 여부가 체의 특성에 의존하기도 합니다.

    정의 5.1.1. 체.

    집합 \(\mathbb{F}\)에 두 개의 이항 연산인 덧셈(\(+\))과 곱셈(\(\cdot\))이 정의되어 있고, 임의의 \(a, b, c \in \mathbb{F}\)에 대하여 다음의 공리들을 만족할 때, (\(\mathbb{F}, +, \cdot\))를 체라고 한다.
    1. 덧셈에 대해 아벨 군을 이룸
      1. 교환법칙: \(a + b = b + a\)
      2. 결합법칙: \((a + b) + c = a + (b + c)\)
      3. 덧셈 항등원 존재: \(\mathbb{F}\) 내에 원소 \(0\)이 존재하여, 모든 \(a\)에 대해 \(a + 0 = a\)를 만족한다.
      4. 덧셈 역원 존재: 각 \(a\)에 대해 원소 \(-a\)가 존재하여, \(a + (-a) = 0\)을 만족한다.
      1. 교환법칙: \(ab = ba\)
      2. 결합법칙: \((ab)c = a(bc)\)
      3. 곱셈 항등원 존재: \(\mathbb{F}\) 내에 0이 아닌 원소 \(1\)이 존재하여, 모든 \(a\)에 대해 \(a \cdot 1 = a\)를 만족한다.
      4. 곱셈 역원 존재: 0이 아닌 모든 \(a \in \mathbb{F}\)에 대해 원소 \(a^{-1}\) (또는 \(1/a\))가 존재하여, \(a \cdot a^{-1} = 1\)을 만족한다.
    3. 분배법칙 (덧셈과 곱셈의 연결): \(a(b + c) = ab + ac\)

    보조설명 5.1.2. 체의 0과 1.

    체의 공리에 \(0 \neq 1\)이 포함되어 있습니다.

    예시 5.1.3. 체의 기초적 예시.

    1. 유리수 체 \(\mathbb{Q}\text{:}\) 두 유리수를 더하거나 곱해도 유리수이며, 0이 아닌 유리수의 역수 역시 유리수입니다.
    2. 실수 체 \(\mathbb{R}\text{:}\) 우리가 가장 흔하게 사용하는 체입니다.
    3. 복소수 체 \(\mathbb{C}\text{:}\) 임의의 0이 아닌 복소수 \(a + bi\)는 역원 \((a-bi)/(a^2+b^2)\)을 가지므로 체가 됩니다. 복소수는 대수적으로 닫혀
       1 
      대수적으로 닫힘:algebraically closed
      있어 고유값(Eigenvalue)을 다룰 때 중요합니다.

    예시 5.1.4. 반례:체가 아닌 구조.

    1. 정수환 \(\mathbb{Z}\text{:}\) 덧셈과 곱셈에 대해 닫혀 있고 교환, 결합, 분배법칙을 모두 만족하지만, 체가 아닙니다. 2, 3과 같은 0이 아닌 원소들이 곱셈에 대한 역원(\(1/2\text{,}\) \(1/3\) 등)을 정수 집합 내에 가지지 않기 때문입니다. 이러한 구조는 환(Ring)이라고 부릅니다.
    2. 자연수 집합 \(\mathbb{N}\text{:}\) 덧셈에 대한 항등원 \(0\)과 역원(음수)조차 존재하지 않으므로 체에서 더욱 거리가 멉니다.
    3. 다항식환 \(\mathbb{F}[x]\text{:}\) \(\mathbb{F}\)가 체일 때, \(\mathbb{F}\)에 계수를 갖는 다항식들의 집합 \(\mathbb{F}[x]\)는 덧셈과 곱셈에 대해 닫혀 있고 우리가 익숙한 대수적 성질들을 만족합니다. 하지만 다항식 \(x\)와 같은 0이 아닌 비상수(non-constant) 다항식들은 곱셈에 대한 역원을 가지지 않습니다. 예를 들어, \(x \cdot p(x) = 1\)을 만족하는 다항식 \(p(x)\)는 존재하지 않습니다(차수를 고려하면 불가능). 따라서 \(\mathbb{F}[x]\)는 체가 아닙니다.
       2 
      이러한 구조는 환(Ring)의 예시입니다만, 환은 본 강의록에서 다루고자 하는 주된 개념은 아닙니다.

    예시 5.1.5. 유한체.

    유한체:finite fields
    체는 반드시 무한히 많은 원소를 가질 필요는 없습니다. 소수 \(p\)에 대하여 정수를 \(p\)로 나눈
    소수:prime numbers
    나머지들의 집합 \(\mathbb{Z}/p\mathbb{Z}\) (또는 \(\mathbb{F}_p\)로 표기)는 모듈로 \(p\) 덧셈과 곱셈에 대하여 체를 이룹니다.
    예를 들어 \(\mathbb{F}_2 = \{0, 1\}\)에서 \(1+1=0\)이며, 이는 컴퓨터 과학과 부호 이론(Coding Theory)에서 가장 핵심적인 역할을 하는 체입니다.

    명제 5.1.6. 영인자의 부재.

    \(\mathbb{F}\)가 체이고 \(a, b \in \mathbb{F}\)일 때, \(ab = 0\)이면 \(a = 0\)이거나 \(b = 0\)이다.

    증명.

    \(ab = 0\)이라고 가정하자. 만약 \(a = 0\)이라면 명제가 성립하므로 증명이 완료된다. 만약 \(a \neq 0\)이라면, 체의 공리에 의해 \(a\)의 곱셈에 대한 역원 \(a^{-1}\)이 존재한다.
    방정식 \(ab = 0\)의 양변의 왼쪽에 \(a^{-1}\)을 곱하면 \(a^{-1}(ab) = a^{-1}(0)\)이다. 결합법칙과 (임의의 원소와 0의 곱은 0)을 적용하면 \((a^{-1}a)b = 0\text{,}\) 즉 \(1 \cdot b = 0\)이므로 \(b = 0\)이다. 따라서 \(a \neq 0\)이면 반드시 \(b = 0\)이어야 하므로 증명이 완성된다.

    예시 5.1.7. 영인자의 예.

    명제 5.1.6가 왜 체의 고유한 성질인지 확인하기 위해, 체가 아닌 합성수 나머지류 \(\mathbb{Z}/6\mathbb{Z}\) = \(\{0, 1, 2, 3, 4, 5\}\)를 생각해 봅시다. 여기서는 \(2 \neq 0\) 이고 \(3 \neq 0\) 이지만, \(2 \times 3 = 6 \equiv 0 \pmod{6}\)이 성립합니다. 즉, \(a \neq 0\text{,}\) \(b \neq 0\)임에도 \(ab = 0\)이 되는 원소(이를 영인자, Zero Divisor라 함)가 존재합니다.
    반면, 소수 \(p = 5\)에 대한 유한체 \(\mathbb{F}_5\)에서는 0이 아닌 두 원소를 곱해서 0을 만들 수 있는 방법이 없습니다. 선형대수학은 스칼라가 체에서 왔다는 사실을 기반으로 하므로, 우리는 \(\mathbb{F}\) 위에서 영인자의 존재를 걱정하지 않아도 됩니다.
    체의 원소가 반드시 수일 필요는 없습니다.

    정의 5.1.8. 유리함수체.

    \(\mathbb{F}\)를 임의의 체라 하고, \(\mathbb{F}[x]\)를 \(\mathbb{F}\)에 계수를 갖는 일변수 다항식환(Polynomial ring)이라 하자. \(\mathbb{F}\) 위의 유리함수체 \(\mathbb{F}(x)\)는 다항식들의 분수들의 집합으로, 다음과 같이 정의된다.
    \(\mathbb{F}(x) = \left\{ \frac{p(x)}{q(x)} \mid p(x), q(x) \in \mathbb{F}[x], q(x) \neq \mathbf{0} \right\}\)
    여기서 \(\mathbf{0}\)은 영다항식을 의미하며, 두 유리식 \(\frac{p_1(x)}{q_1(x)}\)와 \(\frac{p_2(x)}{q_2(x)}\)는 \(p_1(x)q_2(x) = p_2(x)q_1(x)\)가 \(\mathbb{F}[x]\) 안에서 성립할 때 서로 같은 원소로 정의한다.
    유리함수체에서 두 유리식의 덧셈과 곱셈은 다음과 같이 정의된다. 두 유리식 \(\frac{p_1(x)}{q_1(x)}\)와 \(\frac{p_2(x)}{q_2(x)}\)에 대하여:
    덧셈: \(\frac{p_1(x)}{q_1(x)} + \frac{p_2(x)}{q_2(x)} = \frac{p_1(x)q_2(x) + p_2(x)q_1(x)}{q_1(x)q_2(x)}\)
    곱셈: \(\frac{p_1(x)}{q_1(x)} \cdot \frac{p_2(x)}{q_2(x)} = \frac{p_1(x)p_2(x)}{q_1(x)q_2(x)}\)
    이러한 연산들은 유리수의 경우와 유사하게 정의되며, 이들이 체의 공리를 모두 만족하므로 \(\mathbb{F}(x)\)는 실제로 체를 이룬다.
