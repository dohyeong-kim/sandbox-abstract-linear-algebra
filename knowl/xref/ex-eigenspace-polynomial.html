예시 3.1.6. 다항식 공간에서의 고유공간.

공간 \(V = \mathbb{F}[x]_{\le 2}\)에서 선형변환 \(T: V \to V\)를 \(T(p(x)) = x p'(x)\)로 정의합시다. \(\lambda = 1\)에 대한 고유공간 \(E_1\)을 구해봅시다. 어떤 \(p(x) = a_0 + a_1 x + a_2 x^2\)에 대하여 \(x p'(x) = p(x)\)가 성립한다고 하면, 양변을 전개하여 \(a_1 x + 2a_2 x^2 = a_0 + a_1 x + a_2 x^2\)을 얻습니다. 계수를 비교하면 \(a_0 = 0\text{,}\) \(a_1 = a_1\text{,}\) \(2a_2 = a_2\)이므로 \(a_0 = 0, a_2 = 0\)입니다. 조건을 만족하는 다항식은 오직 \(a_1 x\) 형태뿐이므로, 고유공간 \(E_1 = \text{span}\{x\}\)라는 1차원 부분공간이 됩니다.
